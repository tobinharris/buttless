Overview

GET /api/funky/thing Returns information about the thing.
GET /api/funky/stuff Returns a list of stuff.
GET /api/funky/stuff/:id/info Returns the info about the stuff.
POST /api/funky/stuff/:id/info Updates stuff.

Global API's

Funky Thing

GET /api/funky/thing

Returns information about the thing.

Example request:

curl -v  http://192.168.1.3/api/funky/thing

This API resolves to /api/funky/thing/info.json.

Example response:

{"id": 2, "name": "Funky Thing"}

Funky Stuff

GET /api/funky/stuff

Returns a list of stuff.

Example request:

curl -v  http://192.168.1.3/api/funky/stuff

This API resolves to /api/funky/stuff/index.json.

Example response:

[
  {"id": 1, "name": "Funky 1"},
  {"id": 2, "name": "Funky 2"}
]

Using this API

This method is very useful.

GET /api/funky/stuff/:id/info

Returns the info about the stuff.

Paramters

Name Example Description Optional
id 1 The id of the stuff

Example request:

curl -v  http://192.168.1.3/api/funky/stuff/1/info

This API resolves to /api/funky/stuff/:id/info.json.

Example response:

{"id": 2, "name": "Funky 2", "description": "This is funky"}

Using this API

POST /api/funky/stuff/:id/info

Updates stuff.

Paramters

Name Example Description Optional
id 1 The id of the device
name My Stuff The new name of the stuff

Example request:

curl -v \
 --data "name=My Stuff"  \
 http://192.168.1.3/api/funky/stuff/1/info

This API resolves to /api/funky/stuff/:id/info.json.

Example response:

{
  "result": "ok"
}
